Carrden
=======

Carrden inventory tracking web app


Web Stack
---------

    ( D3.js?    )
    [ Bootstrap ]
    [ Scalatra  ]
    [ Slick     ]
    [ Postgres  ] / [ MySQL ]


Build & Run
-----------

```sh
$ cd Carrden_Inventory
$ ./sbt
> container:start
> browse
```

If `browse` doesn't launch your browser, manually open [http://localhost:8080/](http://localhost:8080/) in your browser.
